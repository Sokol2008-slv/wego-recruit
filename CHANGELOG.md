# WEGO Recruit — История разработки и план

## Проект
- **Клиент:** WEGO (рекрутинговое агентство, синие воротнички)
- **Стек:** Next.js 15 + TypeScript + Tailwind CSS + Supabase + Inngest + Vercel
- **Репо:** github.com/Sokol2008-slv/wego-recruit
- **Прод:** https://recruit-starter.vercel.app
- **Админка:** https://recruit-starter.vercel.app/admin

---

## Что сделано (29.03.2026)

### Анкета (13 шагов)
- Возраст, гражданство, разрешение на работу, тип работы (мультивыбор)
- Страна, жильё, начало работы, график, пара, польский язык
- Ограничения (с комментарием), локация (текст)
- Контакты: имя, фамилия, email, телефон (+код страны), Telegram (необязательно)
- Валидация на каждом шаге, кнопка "Назад"
- Данные на русском (не внутренние ключи)

### Вакансии
- Хранятся в Supabase (таблица vacancies)
- Карточки с зарплатой, тегами, жильём, графиком
- Кнопка "Подробнее" → детальная страница → "Подать заявку"
- Фильтр по стране, бейдж "Подходит вам" по анкете кандидата
- Блокировка подачи заявок при активной встрече

### Заявки и дашборд кандидата
- 4 вкладки: На рассмотрении / Одобрено / Отклонено / Встречи
- Из "Одобрено" → "Назначить встречу" → остальные одобренные авто-отклоняются
- После выбора → экран "Отправьте документы на hr@wego-agency.com"
- Вкладка "Встречи" показывает дату, время, статус, документы

### Telegram-бот (@WEGOagency_bot)
- Регистрация кандидата → уведомление рекрутёру с полной анкетой
- Подача заявки → уведомление рекрутёру + работодателю (кнопки: Есть места / Нет мест)
- Одобрение → уведомление рекрутёру
- Назначение встречи → уведомление рекрутёру
- Отмена встречи → уведомление рекрутёру + заявка → "Отклонено"
- Все уведомления с датой/временем (Europe/Warsaw)

### Админ-панель (/admin)
- Вход: email (из списка) + пароль (wego2026)
- Вкладка "Все заявки": таблица с поиском (имя, телефон, email, вакансия, статус)
- Сортировка по дате/имени/статусу, фильтр по статусу
- Контакты в таблице (телефон, email)
- Относительное время ("2ч назад")
- Кнопка "Создать встречу" для заявок со статусом "Выбрано"
- Вкладка "Кандидаты": поиск, карточка с полной анкетой + историей заявок + встреч
- Вкладка "Встречи": список всех встреч, кнопки "Документы ОК", "Назначить дату", "Отменить"
- Статистика: кандидаты, заявки, ожидающие, активные встречи

### Уведомления на сайте
- Колокольчик в навигации
- Показывает новые статусы (одобрено, отклонено, встреча)
- Бейдж с количеством непрочитанных

### Инфраструктура
- **Supabase:** 6 таблиц (candidates, employers, vacancies, applications, meetings, employer_requests)
- **Inngest:** 5 функций (регистрация, заявка, ответ, выбор, напоминание о встрече)
- **Vercel:** авто-деплой, env variables настроены
- **Мобильная версия:** гамбургер-меню, адаптивные таблицы

## Как запустить локально

```bash
git clone git@github.com:Sokol2008-slv/wego-recruit.git
cd wego-recruit
npm install
# Скопировать .env.local (попросить у Георгия)
npm run dev
# Открыть http://localhost:3000
```

## Env variables (нужны для работы)
Попросить у Георгия файл `.env.local` с ключами Supabase, Telegram, Inngest, JWT.
